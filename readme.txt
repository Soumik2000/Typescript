This file contains an workthrough to get started with typescript

To run typescript code we need 3 software to be installed in our system

1. Node
2. Typescript
3. VS code or Visual Studio Code

To install typescript run the following command in the command prompt -> npm install -g typescript

Q. What is typescript?

Ans:    Typescipt refers to the superset of javascript where we can specify type of variables 		and functions which makes it a statically typed language.
	Typescipt provide type safety. Typescript files first transcompiled to javascript 	files. 


	The command to transcompile typescript files is -> tsc filename.ts

	After compilation a javascript file is generated with the same filename.
